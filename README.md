## 🛠️ 如何快速開始？

這是一個「無伺服器、免編譯」的純前端工具，您只需要按照以下步驟即可部署完成：

1. **上傳本專案**：將所有檔案推送到您的 GitHub 儲存庫。
2. **開啟網頁功能**：
   - 進入 GitHub 專案的 `Settings` > `Pages`。
   - 在 `Build and deployment` > `Source` 選擇 **"Deploy from a branch"**。
   - 選取 `main` 分支並選擇 `root` 資料夾，按下 `Save`。
3. **設定 API Key**：
   - 開啟網頁後，點擊右上角的 **「API 設定」**。
   - 貼上您的 [Google AI Key](https://aistudio.google.com/app/apikey)。
   - 您輸入的 Key 會儲存在您的瀏覽器中，不會傳送給任何人。
